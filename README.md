# drowsiness_detection_system
A drowsiness detection system is a technology that monitors a person’s alertness—usually by tracking eye movement, blinking, or head position—to detect signs of fatigue and prevent accidents, especially in drivers.



Installation
1. Clone the repository
  git clone <repository-url>
  cd drowsiness
2. Install dependencies
  pip install opencv-python dlib numpy scipy
3. Make sure required files are present:
  shape_predictor_68_face_landmarks.dat
  Audio file (music.wav)
4.Run
python Drowsiness_Detection.py
